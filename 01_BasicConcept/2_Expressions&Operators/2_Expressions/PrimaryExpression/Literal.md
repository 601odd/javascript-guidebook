# 字面量

字面量（Literal），又名直接量，即程序中直接使用的数据值。

| 数据类型         | 字面量              |
| ---------------- | ------------------- |
| `Null` 类型      | `null`              |
| `Undefined` 类型 | `undefined`         |
| `Boolean` 类型   | `true` `false`      |
| `Number` 类型    | `1`                 |
| `String` 类型    | `"hello"` `'world'` |
| `Regexp` 类型    | `/pattern/`         |
